# ChessGame
Text based chess game.
Rules: Normal Chess Rules + Kill the king to win; No stalemate case.

Player 1 is white, player 2 is black

WK = White King
WQ = White Queen
WB = White Bishop
WN = White Knight
WR = White Rook
WP = White Pawn

BK = Black King
BQ = Black Queen
BB = Black Bishop
BN = Black Knight
BR = Black Rook
BP = Black Pawn
